Project on console


Visual Studios is a coding app where you can type HTML, CSS, and JS. Visual studios can be downloaded and upened up easly. To make your first file drag your cursor onto the left hand side and rightclick. Then select "new file" and name it index.html. Here you can write html code. Make another file and name it style.css, and you can write css code there. same goes to js.


The console screen is used to record on what is happening with the code. To open the console screen, Simply right-click anywhere on the page and press "inspect". This brings up a file of code you have done. On the top hand side of inspect there will be a bar called "console" Click that and you will get to see what has been recorded.
